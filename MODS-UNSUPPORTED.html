# Unsupported Mods and Clients

Not everything that works on PC works on PojavLauncher. Below is a non-exhaustive list of mods that may not work.

# Performance Mods + Shaders

  • Sodium (opens new window) (patch (opens new window) for GL4ES)

  • Rubidium (opens new window) (patch (opens new window) for GL4ES)

  • Chlorine (opens new window)

  • ShadersMod (GLSL Shaders) (opens new window)

  • Iris (opens new window), Iris Installer (opens new window)

  • OptiFine (opens new window) (Render Regions don't work, everything else does) - Big note here. The only supported shaders are Open4ES (opens new window) for GL4ES. They only work on some Qualcomm Snapdragon CPUs and Minecraft 1.8 to 1.16.5. Open4ES supported phones/SoCs below:

    • Poco F3 (Snapdragon 870)
    • Oppo A53 (Snapdragon 460)
    • Xiaomi Redmi Note 8 (Snapdragon 665)

# Skyblock Mods

  • Essential (opens new window)
  • Skytils (opens new window) (1.3.0-pre5 (opens new window) works)
  • Patcher (opens new window) (1.6.1 (opens new window) works)
  • NotEnoughUpdates (opens new window) (2.1.1-pre-1 (opens new window) works)
  • Skyblock Extras (opens new window)

# Input-related Mods

  • MidnightControls (opens new window)
  • Controllable (opens new window)
  • MouseTweaks (opens new window)
  • almost all of them don't work because they aren't made with Android's inputs in mind
