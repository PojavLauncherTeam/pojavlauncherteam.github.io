# Changing your Skin
Hey! Are you tired of seeing that old-fashion Steve/Alex skin in Pojav? Well, we got you covered on how to change skins! Just read this very well.

### You can:
- change it if you don't have an account (offline mode feature)
- change it if you have an account

## Changing skin without an account
Create a texture/resource pack, and make sure that **the .zip file only contains the following: a assets folder and a pack.mcmeta file (ask the discord if what's pack.mcmeta)**

Now, grab your skin and place it in `assets/minecraft/textures/entity/`, then rename it to 'steve.png'

*Still don't know how? You can request the discord community to make it for you, or to give a more in-depth tutorial*

## Changing skin with an account
You can go to [minecraft.net](https://minecraft.net/en-us/profile/skin) and login, then go to your Java profile and change the skin there.
